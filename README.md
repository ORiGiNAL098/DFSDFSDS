# Weather Diary (Дневник погоды)

**Автор:** Богачёв Дмитрий

## Описание

GUI-приложение для ведения дневника погоды. Позволяет добавлять записи о погоде (дата, температура, описание, наличие осадков), фильтровать их по дате и температуре, а также сохранять/загружать данные в формате JSON.

## Как запустить

1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `weather_diary.py`.
3. Запустите в терминале:
   ```bash
   python weather_diary.py
